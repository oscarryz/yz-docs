#example

```js
input String
digits: '0123456789'.split()

n: 0

first: {
  a String
  check: {
     s String
     digits.contains(s) ? {
        a = s
        state = second
     }, { }
  }
}

second: {
   a String
   check: {
     s String
     match {
       digits.contains(s) => {
         a = s
       }
     }, {
       s == '\n' => {
         n = n + int.parse(
           first.a ++ a
         )
         state = first
       }
     }
   }
}
state: first

input.each(state.check)
print("Solution: ${n}")
```
