Other nomenclature: 

Terminology settled (use these terms going forward):
	  
	  #(..)        = boc interface / boc signature
	  #(..) { }    = boc declaration
	  #(..) = { }  = boc expanded form
	  { }          = boc literal
	  name : { }   = short boc declaration (signature inferred, all fields public)
